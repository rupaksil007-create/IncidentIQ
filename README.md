# IncidentIQ – AI Incident Root Cause Analyzer 🚀

**IncidentIQ** is a premium, AI-driven incident analysis platform designed for modern SRE and DevOps teams. It instantly transforms messy system logs into actionable root cause insights using Google Gemini AI.

## 🌟 The Problem
During system outages, engineers waste 70% of their MTTR (Mean Time To Resolution) simply digging through logs across multiple services to find where the failure started. Every minute of downtime costs thousands of dollars.

## ✨ Our Solution: "The AI Incident Copilot"
IncidentIQ acts as a second pair of expert eyes. Paste your logs, and our AI:
1. **Pinpoints the Root Cause** (e.g., Redis memory eviction, K8s OOM, DB deadlock).
2. **Visualizes the Failure Timeline** automatically.
3. **Assigns Confidence Scores** to its findings.
4. **Suggests Immediate Fixes** and long-term prevention strategies.

## 🛠 Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS (Premium Glassmorphism).
- **Animations**: Framer Motion.
- **Backend**: Node.js, Express.
- **AI Engine**: Google Gemini Pro API.
- **Icons**: Lucide React.
- **Data Fetching**: Axios.

## 🚀 Quick Start (Demo Mode)

### Prerequisites
- Node.js (v18+)
- Gemini API Key (Optional for demo data, required for real analysis)

### Installation
```bash
# Clone the repository
git clone <repo-url>
cd IncidentIQ

# Install all dependencies (Root, Server, Client)
npm run install-all
```

### Setup Environment
Create a `.env` file in the `server` directory:
```env
GEMINI_API_KEY=your_api_key_here
```

### Run the App
```bash
npm run dev
```
Visit `http://localhost:5173` to see the landing page.
